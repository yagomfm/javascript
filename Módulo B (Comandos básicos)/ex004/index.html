Formatando strings e números

Strings:

  • s.length -> quantos caracteres a string tem
  • s.toUpperCase() -> tudo para maiuscula
  • s.toLowerCase() -> tudo para minuscula

Números:

  • n.toFixed() -> Configura número de casas decimais
  • n.toFixed().replace('', '') -> Altera um elemento para outro
  • n.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) -> Altera o valor monetário do número
