Operadores

  • Aritméticos
  • Atribuição
  • Relacionais
  • Lógicos
  • Ternários

Operadores Aritméticos:

  • 5 + 2 = 7
  • 5 - 2 = 3
  • 5 * 2 = 10
  • 5 / 2 = 2.5
  • 5 % 2 = 1
  • 5 ** 2 = 25

Operadores de Atribuição Simples:

  • var a = 5 + 3
  • var b = a % 5
  • var c = 5 * b ** 2
  • var d = 10 - a / 2
  • var e = 6 * 2 / d
  • var f = b % e + 4 / e

Auto-atribuições:

  • var a = 3
  • a = a + 4 (a += 4)
  • a = a - 5 (a -= 5)
  • a = a * 4 (a *= 4)
  • a = a / 2 (a /= 2)
  • a = a ** 2 (a **= 2)
  • a = a % 5 (a %= 5)

Incremento:

  • var a = 5
  • a = a + 1 (a ++)
  • a = a - 1 (a --)

Operadores Relacionais:

  • 5 > 2 -> true
  • 7 < 4 -> false
  • 8 >= 8 -> true
  • 9 <= 7 -> false
  • 5 == 5 -> true
  • 4 != 4 -> false

Operadores de Identidade:

  • 5 == 5 -> true
  • 5 == '5' -> true
  • 5 === '5' -> false
  • 5 === 5 -> true

Operadores Lógicos

  • ! -> negação (não)
    • !true -> false (se não é true, então é false)
    • !false -> true (se não é false, então é true)
  • && -> conjunção (e)
    • true && true -> true (os dois tem que ser verdadeiros)
    • true && false -> false
    • false && true -> false
    • false && false -> false
  • || -> disjunção (ou)
    • true || true -> true (basta que um seja verdadeiro)
    • true || false -> true
    • false || true -> true
    • false || false -> false

Operadores Ternários:

  • ? -> o que vai acontecer quando um teste lógico for verdadeiro
  • : -> o que vai acontecer quando um teste lógico for falso
