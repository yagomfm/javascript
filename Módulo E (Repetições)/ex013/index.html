Estruturas de repetição / laços

Estrutura de repetição com teste lógico no início

        
            while(condição) { 
                bloco true
            }
            bloco false
        
        
            var c = 1
            while (c <= 10) {
                console.log(`Passo ${c}`)
                c++
            }

Estrutura de repetição com teste lógico no início

        
            do {
                bloco 1
            } while(condição)
        
        
            var c = 1
            do {
                console.log(`Passo ${c}`)
                c++
            } while (c <= 10)
